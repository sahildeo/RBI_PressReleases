 XSDs, Rules, Sample Messages, revised FAQ w.r.t ISO 20022 standard 
Message formats for Next Generation RTGS (NG-RTGS) System 
The Reserve Bank of India has placed on its website today, ISO 20022 
standard message formats - Rules, XSDs, Mapping Document, API Web Service  
information and Revised FAQs for Next Generation Real Time Gross Settlement 
System (NG-RTGS) for wider dissemination. 
With reference to the earlier press release dated December 28,  2012 on ISO 
200022 Standard Message Formats for the Next Generation Real Time Gross 
Settlement (NG-RTGS) System, the revised messages formats along with FAQs 
were released on  January 31, 2013 after receiving feedback from the Financial 
Institutions, ISO certifying authorities and published on the RBI web-site 
www.rbi.org.in on January 31,  2013 for the benefit of banking/financial Industry using 
the existing link provided.  
The followings documents are now released relating to NG-RTGS project.  
a. 
Information on API for Web Services - (API Web Service v1.1.pdf)
b. 
Message Validation Rules along with Messages (Message with Rules.zip)  
c. 
XSDs for messages formats (RBI NGRTGSXSDs V1 .4.zip)  
d. 
RTGS ISO 20022 Mapping Document (RTGS-ISG 20022 - Message Mappinq v4.0 
August 2013)
e. 
Sample Messages (All Sample Messages 06032013.zip)
f.  
Revised FAQs  (FAQ for ISO20022-26072013.pdf)
g. 
XSD for XML file Upload (XML File Upload.xsd)
h. 
XML File Upload Format for R41 & R42 (XML File Upload Format -R41ANDR42)
i. 
Main IFSC Code List (Main IFSC Details.xls)
j. 
Annex-I: Technical Requirements (Annex I)
k. 
Annex-ll: Member Details (Annex II)
Any queries/clarifications in this regards may please be emailed.
